React
Czym jest i dlaczego wybrać React?

React otwartoźródłowa biblioteka JavaScript stworzona przez Facebooka. Pozwala ona developerom na budowanie dużych aplikacji internetowych, które mogą zmieniać dane, nie przeładowując przy tym strony. Jest wykorzystywana do tworzenia szybkich aplikacji z dynamicznymi interfejsami użytkownika.
